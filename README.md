# 1-uzduotis
V.02 versijoje funkcijos yra išskirtos į atskirus .cpp ir .h failus. Taip pat kai yra nuskaitoma iš failo, įdėtas exception, kad kiekvieną kartą tikrintų ar gerai įvestas failo pavadinimas. Jei ne, išmetama "Neteisingas failo pavadinimas", taip žinome kur palikome klaidą. Pati programa nuskaito duomenis iš duom.txt (bendras failas i kuri kopijavau testus) failo ir sukuria nauja txt faila "rez.txt'. Nepavyko rušiavimas (std::sort kažkodėl neveikia).
